Seis anos após comparar pela primeira vez Java e Go microserviços, os autores revisitam a questão de desempenho: se serviços HTTP cuidadosamente implementados em Java e Go ainda entregam resultados semelhantes nos runtime modernos e hardware. Em vez de apresentá-los como uma competição de linguagem, o artigo de 2026 analisa como cada um desempenha à medida que aumentam as tamanhos de carga e a concorrência, com todo o código, benchmarks e resultados disponíveis em uma repósitório auxiliar.
